IMAGES FOLDER - HOW TO USE
==========================

This folder is for storing local images that you can use in your quiz questions.

SUPPORTED FORMATS:
- JPG/JPEG
- PNG
- GIF
- WEBP

HOW TO ADD IMAGES:
1. Copy your image files into this folder (e.g., santa.jpg, snowman.png)
2. In the Question Manager, select "Local Image" option
3. Choose your image from the dropdown
4. The image will be served at: http://localhost:8000/images/your-image.jpg

EXAMPLE IMAGE NAMES:
- christmas_tree.jpg
- santa_claus.png
- reindeer.gif
- snowflake.webp

TIPS:
- Use descriptive filenames (lowercase with underscores)
- Keep file sizes reasonable (< 2MB recommended)
- You can also use external URLs by selecting "External URL" option
